# Mobile-Sales-Dashboard
This Power BI dashboard provides insights into mobile phone sales, including revenue trends, top-selling products, regional performance, and customer purchase patterns. It helps stakeholders make data-driven decisions using interactive visualizations.

**Features ->**
- Total sales and revenue overview
- Monthly and yearly sales trends
- Top-selling mobile models and brands
- Region-wise sales performance
- Customer segmentation and demographics
- Interactive filters and slicers for detailed analysis
